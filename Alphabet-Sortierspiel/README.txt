# Kurze Erklärung zum Spiel

Das Spiel läuft lokal. Es ist kein Web-Server nötig. Nur ein Browser mit aktivierten Javascript. Interenetzugang ist zum Spielen nicht nötig. Alle benötigten Funktionalitäten stehen in der index.html oder im nebenliegenden data-Ordner.

Um das Spiel zu Starten, öffne index.html. Die Spielanleitung wird im Spiel angezeigt.


Viel Spaß.
